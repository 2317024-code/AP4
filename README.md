# DevPortfolio - Responsive & Interactive Web Project

A modern, responsive developer portfolio website built using HTML, CSS and JavaScript.
This project showcases web development skills through multiple sections, interactive features and mini web apps - all in one page.

# Features

## Portfolio Sections

About Me — Responsive layout with developer intro, image, and skills tags.

Projects — Interactive project cards with hover effects and responsive grid design.

Apps Section — Includes two mini JavaScript apps:

## To-Do List App:

Add and delete tasks dynamically.

Uses localStorage for data persistence (saves tasks even after reload).

## Product Filter & Sort App:

Filter products by category (Electronics, Clothing, Books).

Sort by price or rating.

Dynamically renders products with interactive buttons.

Contact Form — Functional form with basic validation and success alert.

# Technical Implementation

## Frontend

HTML5 for semantic structure.

CSS3 with variables, Grid, and Flexbox for responsive design.

JavaScript (ES6) for interactivity and DOM manipulation.

## Functionality

Responsive navigation with hamburger menu for mobile.

Smooth scroll navigation between sections.

LocalStorage integration for persistent To-Do data.

Dynamic rendering for product grid filtering and sorting.

## Responsiveness

Fully responsive design that adapts to all devices (mobile, tablet, desktop).

Uses media queries and flexible layouts for smooth scaling.

## Learnings

Combining layout (CSS) and logic (JS) in a single-page app.

Implementing data persistence using localStorage.

Creating reusable UI components (cards, buttons, sections).

Enhancing UX through micro-animations and hover transitions.

## Tech Stack

Languages: HTML, CSS, JavaScript

Tools: VS Code, Git, Browser DevTools
